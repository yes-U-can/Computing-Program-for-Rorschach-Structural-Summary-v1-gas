# Computing Program for Rorschach Structural Summary (v1-gas)

Legacy Google Apps Script (GAS) implementation of the Rorschach Structural Summary calculator.

## Overview

- This repository contains the original v1 web app built with Google Apps Script.
- It was published to share practical scoring and automation ideas with clinicians and researchers.
- The current production-focused v2 (Next.js) app is maintained separately.

## Included Files

- `Code.gs`: GAS server-side script
- `index.html`: Main UI
- `styles.html`: UI styles

## Tech Stack

- Google Apps Script (JavaScript)
- HTML / CSS

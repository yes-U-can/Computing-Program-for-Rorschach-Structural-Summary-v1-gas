# 로샤 구조 요약지 계산 프로그램 (Computing Program for Rorschach Structural Summary)

수기로 계산하던 로샤 검사의 구조 요약지를 자동으로 계산하고 출력할 수 있도록 만든 Google Apps Script 기반 웹 애플리케이션입니다.

## 프로젝트 소개

로샤 검사 결과 자동화를 위한 시도는 다양한 형태(exe 프로그램, Excel 매크로 등)로 이어져 왔습니다.
이 프로젝트는 **Google Apps Script(GAS)** 를 사용해, 별도 설치 없이 웹에서 바로 사용할 수 있는 형태로 구현한 버전입니다.

임상 현장에서 더 편리하게 활용할 수 있도록 공개했으며, 오류나 개선 의견은 언제든 환영합니다.

본 프로그램의 저작권은 **서울임상심리연구소(SICP)** 에 귀속되어 있으나,
임상심리학 분야의 발전과 실무 지원을 위한 공익적 목적에서 코드를 공개합니다.

## 주요 기능

1. **자동 계산**: 반응 영역, 내용, 결정인 등 채점 항목 자동 계산
2. **언어 반응 메모**: 피검자의 주요 언어 반응 기록 및 관리
3. **결과지 출력**: 최종 구조 요약지 인쇄용 출력 지원

## 사용 기술

- Google Apps Script (JavaScript)
- HTML / CSS
